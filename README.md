# Smart SMS

A NodeJS app that is used to manage SMS messages and contacts

# Setup

- $ git clone https://github.com/micahoriaso/smart-sms.git
- $ cd smart-sms
- $ npm install
- $ npm start

# Technologies

- NodeJS
- ExpressJS
- MongoDB
- Mongoose
